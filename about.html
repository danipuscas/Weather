Despre Aplicația de Vreme

Bine ați venit la aplicația noastră de vreme! Această aplicație vă oferă informații actualizate despre vremea curentă și previziuni pentru diferite locații.

Cum funcționează?

Aplicația utilizează o combinație de date în timp real și servicii de prognoză meteorologică pentru a vă oferi informații precise despre vreme. Puteți căuta locații specifice și veți primi informații despre temperatura actuală, umiditate, vânt și alte detalii relevante.

Cum să utilizați aplicația?

Pentru a utiliza aplicația, pur și simplu introduceți numele locației în bara de căutare și apăsați butonul "Caută". Aplicația va afișa apoi informațiile despre vremea curentă și previziunile pentru locația respectivă.

Cod JavaScript pentru obținerea informațiilor despre vreme

Acest cod JavaScript este parte dintr-o aplicație de vreme care utilizează API-ul OpenWeatherMap pentru a obține informații meteorologice în timp real pentru un anumit oraș. În primul rând, el definește URL-ul de bază și cheia API necesară pentru a face cereri către serviciul OpenWeatherMap. Apoi, el selectează elementele HTML relevante pentru căutarea și afișarea informațiilor despre vreme. Funcția CumEVremea(city) este responsabilă pentru a obține datele despre vreme pentru un oraș dat. Ea face o cerere la API, așteaptă răspunsul și actualizează apoi elementele HTML cu datele obținute. Iconița vremii este schimbată în funcție de condițiile meteorologice. Dacă orașul introdus nu poate fi găsit, utilizatorul este notificat printr-o alertă.

  • const searchBox = document.querySelector(".search input");: Această linie selectează elementul input din cadrul clasei "search". Acest element va fi utilizat pentru a introduce numele orașului pentru care se dorește informații despre vreme.
  • const searchButton = document.querySelector(".search button");: Aici se selectează butonul din cadrul clasei "search", care va fi utilizat pentru a declanșa căutarea informațiilor despre vreme atunci când este apăsat.

  • async function CumEVremea(city) : Această linie definește o funcție asincronă numită "CumEVremea", care primește ca argument numele orașului pentru care se dorește informații despre vreme.

  • document.querySelector(".weather").style.display = "block";: Aici se face ca div-ul cu clasa "weather" să fie afișat, pentru a arăta informațiile despre vreme.
